## Demo

https://back-roads-sarker.netlify.app/

## Run Locally

Clone the project

```bash
  git clone https://github.com/sarker25/back-roads.git
```

Go to the project directory

```bash
  cd back-roads
```

Install dependencies

```bash
  yarn
```

Start the server

```bash
  yarn dev
```
